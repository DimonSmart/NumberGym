# Brand Style Guide

Core palette (5 colors):

1. Primary Blue: `#72BFE6`
2. Deep Blue: `#26679D`
3. Brand Gold: `#FEB60B`
4. Warm Orange: `#E37E21`
5. Ice Background: `#F3F9FE`

Usage:
- Primary Blue: icons, primary accents, active highlights.
- Deep Blue: pressed states, stronger contrast accents.
- Brand Gold: branding, rewards, key callouts.
- Warm Orange: secondary emphasis.
- Ice Background: default app canvas.
