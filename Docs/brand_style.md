# Brand Style Guide

Here’s a compact **5-color brand palette** for NumbersGym (with **2 blues**):

1. **Primary Blue (Character Blue)** – `#72BFE6`
   *Use for primary accents, icons (including your menu button), highlights, friendly UI emphasis.*
   *Similar to: Sky Blue / Light Sky Blue*

2. **Deep Blue (Support Blue)** – `#26679D`
   *Use for pressed states, stronger contrast accents, outlines/shadows related to the blue theme.*
   *Similar to: Steel Blue*

3. **Brand Gold (Logo Gold)** – `#FEB60B`
   *Use for the main brand accent (logo “Numbers”), key callouts, badges, rewards.*
   *Similar to: Gold / Goldenrod*

4. **Warm Orange (Secondary Accent)** – `#E37E21`
   *Use for secondary highlights, energy cues, small emphasis details (pairs well with gold).*
   *Similar to: Dark Orange*

5. **Ice Background (App Background)** – `#F3F9FE`
   *Use as the main background / canvas color, gives the clean gym-sky feel.*
   *Similar to: Alice Blue*
